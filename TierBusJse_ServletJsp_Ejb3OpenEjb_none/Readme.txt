INSTALLINT EJB ON NOT JEE SERVER


If you want to use Ejb on not jee server (like tomcat or jetty) you have to 
first install ejb there. 

To install ejb implementation (OpenEjb) on not jee server you have to:
- download OpenEjb war file (For instance: tomee-webapp-1.5.1.war);
- deploy on Tomcat 7;
- restart Tomcat 7;
